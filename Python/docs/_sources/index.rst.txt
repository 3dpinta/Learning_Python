Welcome to Fundamentals of Python's documentation!
===================================================
In this module, you will:

* Learn what the Python programming language is and what a Python interpreter is.
* Install Python and related tools/packages on your computer via the distribution Anaconda.
* Familiarize yourself with Jupyter notebook - a valuable tool for tinkering with Python code.
* Work through an informal introduction of Python.
* Work through an informal introduction of NumPy - the fundamental package for scientific computing in Python.
